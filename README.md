# 🐍 Python Projects

A collection of Python programs covering core programming concepts, algorithms, data structures, and automation fundamentals.

## 🛠️ Technologies Used
- **Language:** Python 3
- **Concepts:** Functions, OOP, List/Array manipulation, Sorting algorithms, Lambda, Map & Filter

## 📁 Projects Included

| File | Description |
|---|---|
| `AgeCalculator.py` | Calculates age from date of birth input |
| `CreditCardValidator.py` | Card number validation using the Luhn algorithm |
| `array_sorting.py` | Bubble sort and comparison-based sorting implementations |
| `transaction_log.py` | Logs and tracks financial transactions |
| `christmas_song.py` | Generates the 12 Days of Christmas using loops |
| `combine_list.py` | List merging and manipulation utilities |
| `function_class.py` | Demonstrates functions, classes, and OOP in Python |
| `vowel&consonants.py` | String parsing to count vowels and consonants |
| `first_last_two.py` | Array slicing and element access patterns |
| `getarray_even.py` | Filters even numbers from arrays using various methods |

## 🚀 How to Run
```bash
# Clone the repository
git clone https://github.com/Officialvalz/python-projects.git

# Run any script
python3 script_name.py
```
